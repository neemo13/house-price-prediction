# House Price Prediction - Linear Regression

This is a beginner machine learning project that predicts house prices using a Linear Regression model.

The project demonstrates basic data preprocessing, handling missing values, model training, and evaluation using Python.

## Technologies Used

- Python
- pandas
- numpy
- scikit-learn
- JupyterLab

## Dataset

The dataset contains various features related to houses such as:

- Lot Area
- Building Type
- Overall Condition
- Year Built
- Basement Area
- Zoning Type

The target variable is:

SalePrice

## Project Workflow

1. Load dataset using pandas
2. Explore dataset structure
3. Handle missing values
4. Select relevant features
5. Split dataset into training and testing sets
6. Train Linear Regression model
7. Make predictions
8. Evaluate model performance

## Model

The project uses **Linear Regression** from scikit-learn.

## Evaluation Metrics

The following metrics are used:

- R² Score
- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- Mean Absolute Percentage Error (MAPE)

Example result from the notebook:

R² Score: ~0.758  
Accuracy: ~75.89%
